# KorPIX 파일럿 유스케이스

## UC-001 소액 정기결제 자동화

**목표**: 월정액 구독료 등 반복 결제를 AI가 자동 처리

**핵심 검증**: ActionRecord 표준 구조 · 기본 Policy Engine

**Risk Score 계산**:
- 금액 구간: 1만 미만 +0, 5만 미만 +5, 10만 미만 +10, 50만 미만 +20, 50만 이상 +35
- 건당 한도 초과: +30
- 넷플릭스 17,000원 → Score 20 → **AUTO_APPROVE**

## UC-002 AI 투자 자동화

**목표**: AI 에이전트가 포트폴리오 전략에 따라 자동 투자

**핵심 검증**: 서킷 브레이커 · 손실 한도 · 적합성 원칙 · 자본시장법 5년 감사

**서킷 브레이커**: VIX ≥ 35 또는 코스피 -5% 이상 → 모든 투자 즉시 DENY

## UC-003 기업 구매 승인 자동화

**목표**: 기업 내부 구매 프로세스를 AI가 처리, 직위별 결재 체인 자동 구성

**핵심 검증**: 조직 계층 결재 · ERP 연동 · 5단계 티어 · 예산 실시간 확인

**승인 티어**:
- Tier 1 (100만 미만): Auto Approve
- Tier 2 (100만~500만): 팀장 승인
- Tier 3 (500만~2,000만): 팀장 → 재무팀
- Tier 4 (2,000만~1억): 팀장 → 재무팀 → CFO
- Tier 5 (1억 초과): DENY (이사회 이관)
